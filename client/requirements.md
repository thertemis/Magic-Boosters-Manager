## Packages
framer-motion | Essential for card flip animations and smooth UI transitions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
- The app uses a dark fantasy aesthetic inspired by Magic: The Gathering.
- Card images will be loaded from Scryfall URLs provided by the backend.
- "Flipping" cards requires 3D transform styles in CSS/Motion.
- Authentication is handled via standard JSON POST requests to /api/auth.
